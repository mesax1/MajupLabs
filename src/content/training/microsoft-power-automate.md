## El Poder del Ecosistema Microsoft

Si tu empresa usa Microsoft 365, tienes acceso a herramientas de automatización de clase empresarial. Power Automate y Copilot transforman la forma en que trabajas.

## ¿Por qué Power Automate?

Microsoft ha invertido fuertemente en su plataforma de automatización:

- **Integración nativa** con todo el ecosistema Microsoft 365
- **RPA incluido** para automatizar aplicaciones de escritorio
- **AI Builder** para IA sin código
- **Copilot** para productividad asistida por IA
- **Enterprise-ready** con gobernanza y seguridad

## ¿Qué Aprenderás?

### Power Automate Cloud

Crea flujos que conectan tus aplicaciones de Microsoft 365. Desde automatizaciones simples hasta procesos de negocio complejos con múltiples aprobadores.

### Power Automate Desktop

La automatización robótica de procesos (RPA) de Microsoft. Automatiza aplicaciones de escritorio, incluso las que no tienen API, grabando tus acciones.

### Microsoft Copilot

La IA de Microsoft integrada en tus herramientas de trabajo. Aprende a sacarle el máximo provecho en Excel, Word, Outlook y Teams.

### AI Builder

Crea modelos de inteligencia artificial sin escribir código. Procesamiento de documentos, predicción, clasificación y más.

## Sistemas que Construirás

| Sistema | Descripción |
|---------|-------------|
| **Document Processor** | Extrae datos de facturas y contratos con AI Builder |
| **Approval Workflow** | Flujo de aprobación multi-nivel con notificaciones |
| **SharePoint Automation** | Gestión automatizada de documentos y permisos |
| **Teams Bot** | Bot que responde preguntas y ejecuta tareas |
| **Legacy App Bridge** | Conecta aplicaciones antiguas con el cloud vía RPA |

## Metodología

Enfoque práctico y progresivo:

1. **Fundamento** (15 min): Concepto y casos de uso
2. **Demostración** (20 min): Construcción paso a paso
3. **Práctica** (45 min): Tu turno de construir
4. **Caso real** (10 min): Aplicación a tu contexto

## Requisitos Previos

- Acceso a Microsoft 365 (licencia business o superior)
- Familiaridad con Excel, Outlook y Teams
- No se requiere experiencia en programación
- Licencia de Power Automate Premium para algunas funcionalidades
