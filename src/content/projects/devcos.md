## Project Overview

Devcos International is a trusted family-owned business with over 60 years of global experience in the health and beauty industry. Operating across Australia, USA, China, and Europe, they provide turnkey services for formula development, packaging, branding, warehousing, and logistics. We developed an AI-powered platform to automate their artwork compliance verification process, ensuring product labels meet stringent legal and regulatory requirements.

## Key Contributions

### AI-Powered Compliance Verification System
- Architected and implemented a full-stack application with FastAPI backend and React frontend for automated artwork compliance checking
- Integrated Azure Document Intelligence for extracting text and layout information from product label artwork files
- Developed AI agent system using LLMs to verify compliance against regulatory rules for health and beauty products
- Built document processing pipeline handling PDFs and images with automatic text extraction and coordinate-based element detection

### Production-Ready Platform Development
- Designed and implemented PostgreSQL database schema with SQLModel ORM for managing artworks, rules, users, and verification results
- Created authentication and authorization system with JWT tokens and role-based access control
- Implemented database migrations using Alembic for version-controlled schema changes
- Built RESTful API with structured logging, health checks, and monitoring endpoints
- Developed responsive React frontend with Material-UI components for file upload, result visualization, and user management

### DevOps and Deployment
- Containerized both frontend and backend applications using Docker with multi-stage builds
- Configured DigitalOcean App Platform deployment with Container Registry integration
- Implemented CI/CD pipeline with GitHub Actions for automated testing, building, and deployment
- Set up production-ready nginx configuration with security headers, gzip compression, and health checks

## Impact

The automated compliance verification platform transformed Devcos's quality assurance process by reducing manual artwork review time from hours to minutes. The system processes artwork files containing numerous pages of documents, automatically identifies compliance issues, and generates detailed reports. This enabled faster time-to-market for clients while maintaining rigorous regulatory standards across multiple countries and product categories.

## Resumen del Proyecto

Devcos International es una empresa familiar de confianza con más de 60 años de experiencia global en la industria de salud y belleza. Operando en Australia, Estados Unidos, China y Europa, brindan servicios completos para desarrollo de fórmulas, empaque, branding, almacenamiento y logística. Desarrollamos una plataforma impulsada por IA para automatizar su proceso de verificación de cumplimiento de artes gráficos, asegurando que las etiquetas de productos cumplan con estrictos requisitos legales y regulatorios.

## Contribuciones Clave

### Sistema de Verificación de Cumplimiento con IA
- Arquitecturamos e implementamos una aplicación full-stack con backend FastAPI y frontend React para verificación automática de cumplimiento de artes gráficos
- Integramos Azure Document Intelligence para extraer texto e información de diseño de archivos de arte gráfico de etiquetas de productos
- Desarrollamos sistema de agentes de IA usando LLMs para verificar cumplimiento contra reglas regulatorias para productos de salud y belleza
- Construimos pipeline de procesamiento de documentos manejando PDFs e imágenes con extracción automática de texto y detección de elementos basada en coordenadas

### Desarrollo de Plataforma Lista para Producción
- Diseñamos e implementamos esquema de base de datos PostgreSQL con SQLModel ORM para gestionar artes, reglas, usuarios y resultados de verificación
- Creamos sistema completo de autenticación y autorización con tokens JWT y control de acceso basado en roles
- Implementamos migraciones de base de datos usando Alembic para cambios de esquema con control de versiones
- Construimos API RESTful con logging estructurado, health checks y endpoints de monitoreo
- Desarrollamos frontend React responsivo con componentes Material-UI para carga de archivos, visualización de resultados y gestión de usuarios

### DevOps y Despliegue
- Containerizamos aplicaciones frontend y backend usando Docker con builds multi-etapa
- Configuramos despliegue en DigitalOcean App Platform con integración de Container Registry
- Implementamos pipeline CI/CD con GitHub Actions para pruebas, construcción y despliegue automatizados
- Configuramos nginx listo para producción con headers de seguridad, compresión gzip y health checks

## Impacto

La plataforma automatizada de verificación de cumplimiento transformó el proceso de aseguramiento de calidad de Devcos reduciendo el tiempo de revisión manual de artes de horas a minutos. El sistema procesa archivos conteniendo numerosas páginas de documentos, identifica automáticamente problemas de cumplimiento y genera reportes detallados. Esto permitió un tiempo de lanzamiento al mercado más rápido para los clientes mientras se mantienen estándares regulatorios rigurosos en múltiples países y categorías de productos.
